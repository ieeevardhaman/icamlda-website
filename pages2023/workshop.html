1st International Workshop on Artificial Intelligence and Data Analytics (AIDA-2023)


Objectives
Data analytics is a critical component of many AI applications. AI algorithms rely on large amounts of high-quality data to learn and make predictions or decisions. Data analytics, on the other hand, involves analyzing and interpreting data to extract meaningful insights. In the context of AI, data analytics is used to preprocess and clean data, and select the most relevant data for the AI algorithm to learn from. After the AI algorithm has been trained, data analytics can also be used to analyze the results and identify the areas for improvement.
AI can also be used to enhance data analytics. For example, AI can be used to automate data cleaning and pre-processing to identify patterns and anomalies in data and to make predictions based on historical data. In summary, AI and data analytics are highly interdependent fields that work together to create intelligent systems that can analyze data, make predictions, and take action based on insights gained from that data. The ultimate goal is to use AI and data analytics to create innovative applications that can solve real-world problems and create value for businesses and individuals.

Topics of interests:
We invite the submission of original works that is related, but are not limited, to the topics below:

  • Machine learning
  • Natural language processing (NLP)
  • Computer vision
  • Robotics
  • Reinforcement learning
  • Descriptive analytics
  • Predictive analytics
  • Prescriptive analytics
  • Data mining
  • Text analytics
